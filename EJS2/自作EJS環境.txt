構造

srcフォルダにある。
ejsファイル（ヘッダー、メイン、フッターに3分割したファイルがそれぞれある）を
include()を用いてindex.ejsに集合させ、コンパイル後
destフォルダの方に
それぞれのejsファイルの内容が集合したhtmlファイルとして書き出す。

package.jsonの"scripts"にて
使用するパッケージ、対象ファイル、出力先を指定している。


同じフォルダにある、scssファイル（レイアウトやパーツ用に分ける為複数ある）を
style.scssファイルに@useを用いて集合させ、コンパイル後、
destフォルダの方に
それぞれのscssファイルの内容が集合したcssファイルとして書き出す。

最終htmlファイルからcssファイルを読み込めば「html＋css」の内容が
表示出来る。

・「npm run ejs」ejsファイルをhtmlファイルにする。
・「npm run sass」scssファイルをcssファイルにする。
・「npm run start」両方を行う。

ブラウザへの表示はVScodeからなら表示出来る。
